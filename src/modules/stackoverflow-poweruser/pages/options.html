StackOverflow Power User
Stop wasting time while programming !

Settings

Sidebar


Shows a floating sidebar on the left that shows score of first 5 answers, as sometimes it can be interesting to read other well upvoted answers. It will also inform you if you have already upvoted an answer previously on this page. You can also hover the first cell to get additional information about the current page. (Try it)

Indicators


If a question has a better answer than the accepted one, it shows an clickable arrow to easily notice it and avoid wasting time. (Try it)


If a question has no answer, it shows a disappointed smiley to easily notice it and avoid wasting time reading the question. (Try it)

Navigation


Automatically scroll to the first answer, as you usually don't have to read the question. (Try it)


Give the ability to navigate through all answers without having to remove your hands from keyboard. (Try it)

Miscellaneous


You need to have at least 1000 reputation on StackOverflow for this feature to work. When arriving on Stack Overflow page, it automatically expand votes count (upvotes/downvotes) and shows the score of the answer (in percent). (Try it)


Make vote buttons scroll so that they are still visible even on long posts. (Try it)


Automatically expand all "show X more comments" when you use CMD/Ctrl + F to search in page. This way you ensure that you're not missing any result. (Try it)
